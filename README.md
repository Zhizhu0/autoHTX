# 火币自动交易


## 项目介绍

无

## 依赖

如果使用打包后的exe文件，则无需任何依赖，直接执行即可。

使用python环境运行，先执行pip install -r requirements.txt安装依赖。

## 使用说明

### 第一次使用

第一次使用前需要生成私有注册码，在命令行中输入python main.py --generate_key或者如果使用exe打包文件执行main.exe --generate_key。

此操作会在数据库中生成一个注册校验码，用于校验用户注册请求是否合法。

该命令一般只使用一次，每次执行时会清除所有已有的用户数据，请谨慎执行。

### 注册账号

再执行python main.py --generate_register_code或者如果使用exe打包文件执行main.exe --generate_register_code。

此操作会输出一个注册码，复制该注册码。

执行python main.py或者直接打开main.exe文件。

在浏览器中输入127.0.0.1:8000，并回车访问，会自动跳转到登录界面，点击注册按钮。

输入要注册的账号和密码，以及刚刚生成的注册码。

注册成功后会自动跳转到登录界面。

一个注册码只能注册一个账号。

### 日常使用

在浏览器中输入127.0.0.1:8000，如果没有登录，则会跳转到登录页面，登录后自动跳转到主页。

参数设置完成后打开自动交易按钮，即使关闭浏览器，也会继续运行。

在查看系统日志中查看运行记录和请求ai立即分析。

在查看提示词和图表中可以预览给ai发送的请求内容和图表。

### 系统更新

在1.1版本之后，如果是python环境，直接执行python main.py --update即可自动更新。

### 注意事项

注意，1.0版本之后数据表结构大改，如果之前有旧版本数据，请删除旧版本数据后再运行，否则程序无法正常运行。数据库保存在用户目录下的.db文件中。
